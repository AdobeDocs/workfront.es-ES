# Conexión de datos de Workfront para Workfront Planning

Workfront Data Connect proporciona acceso a los datos de Workfront, incluso de Adobe Workfront Planning, mediante el uso de la tecnología de Snowflake, lo que le permite responder preguntas críticas para la empresa y medir el valor de sus procesos. Los clientes empresariales pueden analizar sus datos con herramientas de visualización de datos directamente desde un lago de datos de Workfront o exportar datos de Workfront a almacenes de datos propiedad del cliente.

Los suscriptores podrán conectar la herramienta de inteligencia empresarial que elijan (Tableau, Power BI, Domo, Looker, etc.) Vaya a Data Connect para crear visualizaciones de datos con los siguientes tipos de datos:

* Baja latencia, datos de estado actual
* Cambio de registros de eventos para comparaciones de momentos concretos
* Instantáneas históricas diarias para el análisis de tendencias

Para obtener información sobre cómo crear una cuenta de Reader con Snowflake y establecer una conexión con Workfront Data Connect, consulte los siguientes artículos:

* [Crear una cuenta de lector o una conexión para Snowflake](/help/quicksilver/reports-and-dashboards/data-lake/create-a-reader-account.md)
* [Establecimiento de una conexión con Workfront Data Conect](/help/quicksilver/reports-and-dashboards/data-lake/share-data-externally.md)


Para obtener información acerca de Planning en el diccionario de datos de Workfront Data Connect, consulte [Diccionario de datos de Workfront Data Connect](/help/quicksilver/reports-and-dashboards/data-lake/data-dictionary.md).
