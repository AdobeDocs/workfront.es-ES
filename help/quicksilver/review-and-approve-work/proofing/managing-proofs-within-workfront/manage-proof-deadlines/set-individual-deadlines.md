# Establecer fechas límites para revisores individuales

Puede establecer fechas límites diferentes para revisores diferentes en una prueba que ya haya creado. Esto requiere la creación de una fase independiente para cada revisor para el que desee establecer una fecha límite individual.

## Establecer fechas límites para revisores individuales

1. Pase el ratón sobre la revisión y luego haga clic en **Flujo de trabajo de revisión** para abrir la página Flujo de trabajo.
1. En la esquina superior derecha, haga clic en **Nueva etapa**

   ![Nuevo_botón_fase.png](assets/new-stage-button.png)

   >[!NOTE]
   >
   >Si no ve la opción nueva etapa, primero debe convertir el flujo de trabajo a un flujo de trabajo automatizado.

1. Añada información para la fase, especialmente el nombre del revisor y la fecha límite que desee para el revisor.
1. Repita los dos pasos anteriores según sea necesario para agregar otros revisores que necesiten su propia fecha límite.
