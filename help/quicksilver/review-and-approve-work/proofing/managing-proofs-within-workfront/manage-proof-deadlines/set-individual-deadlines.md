# Establecer fechas límites para revisores individuales

Puede establecer fechas límites diferentes para revisores diferentes en una prueba que ya haya creado. Esto requiere la creación de una fase independiente para cada revisor para el que desee establecer una fecha límite individual.

## Establecer fechas límites para revisores individuales

1. Pase el puntero por encima de la revisión y luego haga clic en **Flujo de trabajo de revisión** para abrir la página flujo de trabajo.
1. En la esquina superior derecha, haga clic en **Nueva fase**

   ![New_Stage_button.png](assets/new-stage-button.png)

   >[!NOTE]
   >
   >Si no ve la opción de nueva fase, primero debe convertir el flujo de trabajo a un flujo de trabajo automatizado.

1. Añada información para la fase, especialmente el nombre del revisor y la fecha límite que desee para el revisor.
1. Repita los dos pasos anteriores según sea necesario para añadir otros revisores que necesiten su propia fecha límite.
