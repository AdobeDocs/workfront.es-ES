# Claves en Adobe Workfront Fusion

El llavero de [!DNL Adobe Workfront Fusion] le ayudará a administrar claves públicas y privadas. Las claves se utilizan, por ejemplo, mediante la variable [!UICONTROL Cifrado] aplicación para cifrar o descifrar mensajes PGP.

## Uso de claves en [!DNL Workfront Fusion]

Si un módulo requiere una clave pública o privada, puede agregarla al llavero. Para obtener más información, consulte [Configure las opciones de un módulo en [!DNL Adobe Workfront Fusion]](../../workfront-fusion/modules/configure-a-modules-settings.md).


Para obtener más información sobre claves públicas y privadas, consulte [Términos básicos en [!DNL Adobe Workfront Fusion]](../../workfront-fusion/get-started/basic-terms.md).
