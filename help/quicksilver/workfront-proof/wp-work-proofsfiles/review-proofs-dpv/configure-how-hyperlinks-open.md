# Configuración de apertura de hipervínculos en el Visor de corrección de escritorio

>[!IMPORTANT]
>
>Este artículo hace referencia a la funcionalidad del producto independiente [!DNL Workfront Proof]. Para obtener información sobre la revisión dentro de [!DNL Adobe Workfront], consulte [Revisión](../../../review-and-approve-work/proofing/proofing.md).

Algunos vínculos del contenido web interactivo están configurados para abrirse en una nueva pestaña o ventana del explorador. Puede configurar [!DNL Workfront] para que abra todos estos vínculos dentro de Desktop Proofing Viewer, lo que permitirá a los revisores revisar el contenido recién abierto. Para obtener más información, consulte [Configuración de ajustes del visualizador de revisión](../../../review-and-approve-work/proofing/reviewing-proofs-within-workfront/configure-proofing-viewer-settings.md).
