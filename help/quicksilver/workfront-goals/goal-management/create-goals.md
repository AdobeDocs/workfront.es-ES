Tanto si es un CEO, un administrador o un colaborador individual, puede crear objetivos en objetivos de Adobe Workfront para alinear su trabajo con sus objetivos y los objetivos que describen la estrategia de su organización.

Debe tener lo siguiente para poder iniciar:

* Plantilla de diseño que incluye el área Objetivos del menú principal.

## Directrices para la creación de objetivos

Antes de comenzar con los objetivos de Workfront, le recomendamos que lea nuestras recomendaciones y directrices de prácticas recomendadas para administrar los objetivos de forma eficaz. Para obtener más información sobre las directrices para la creación y administración de objetivos, consulte [Información general sobre los objetivos de Adobe Workfront](../../workfront-goals/goal-management/wf-goals-overview.md).

## Crear objetivos

En este artículo se describe cómo crear un objetivo estratégico en los objetivos de Workfront. Para obtener información sobre la creación de un objetivo de caso empresarial, consulte [Crear objetivos de caso empresarial](../../manage-work/projects/define-a-business-case/create-business-case-goals.md).

Puede crear un objetivo estratégico de una de las siguientes maneras:

* [Crear un objetivo desde cero](#create-a-goal-from-scratch)
* [Copiar un objetivo existente](#copy-an-existing-goal)
* [Conversión de un resultado o actividad en un objetivo](#convert-a-result-or-activity-to-a-goal)

1. Introduzca información en los campos siguientes:
   * **Nombre del objetivo**: Escriba un nombre para el objetivo. Este campo es obligatorio.
   * **Periodo**: Seleccione un trimestre o año predefinido de la variable **Periodo** campo desplegable

      O

      Seleccione el **Habilitar fechas personalizadas** a continuación, seleccione una **Inicio** y **Fecha final** para el objetivo.

   * **Propietario del objetivo**: Empiece a escribir el nombre de un usuario, un equipo, un grupo o una organización para indicar quién es el propietario del objetivo. Se selecciona como propietario del objetivo de forma predeterminada.

   El nuevo objetivo se enumera en la lista de Objetivos y tiene el estado de **Borrador**.

   Debe asociar un objetivo con un indicador de progreso para activarlo y comenzar a trabajar en él.

   Realice al menos una de las siguientes acciones para preparar un objetivo que se activará:
   * Agregar un resultado

      Para obtener información sobre cómo agregar resultados, consulte [Agregar resultados a objetivos en Objetivos de Adobe Workfront](../results-and-activities/add-results-to-goals.md).
   * Agregar una actividad

      Para obtener información sobre cómo agregar actividades, consulte [Agregar actividades a objetivos en Objetivos de Adobe Workfront](../results-and-activities/add-activities-to-goals.md).
   * Alinear otro objetivo con él

      Para obtener información sobre la alineación de objetivos, consulte [Alinear objetivos conectándolos en los objetivos de Adobe Workfront](../goal-alignment/align-goals-by-connecting-them.md).


### Copiar un objetivo existente {#copy-an-existing-goal}

Puede crear un objetivo copiando uno existente.

Para obtener información sobre cómo copiar objetivos, consulte [Copiar objetivos en objetivos de Adobe Workfront](../../workfront-goals/goal-management/copy-goals.md).

### Conversión de un resultado o actividad en un objetivo {#convert-a-result-or-activity-to-a-goal}

Puede crear un objetivo convirtiendo el resultado o la actividad de un objetivo existente en un objetivo. El nuevo objetivo se alinea con el objetivo original.

Para obtener información sobre cómo convertir resultados y actividades en objetivos, consulte [Alinear objetivos convirtiendo resultados y actividades en objetivos](../../workfront-goals/goal-alignment/align-goals-by-converting-results-activities.md).
