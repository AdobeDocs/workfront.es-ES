# Cambiar el formato de las fechas en [!DNL Adobe Workfront]

<!--this article used to be called "Change the date format in Adobe Workfront when using Chrome". The team decieded to make it more generic and hide the steps. Also see drafted content below-->

Cambiar el formato de las fechas en [!DNL Adobe Workfront], como [!UICONTROL Fecha de finalización planeada], [!UICONTROL Fecha de finalización real]o [!UICONTROL Fecha de finalización prevista], de DD/MM/AAAA a MM/DD/AAAA o viceversa, por ejemplo, debe cambiar la configuración de idioma en el navegador.

Si el idioma predeterminado del explorador es *[!UICONTROL Inglés (Estados Unidos)]*, las fechas se muestran en formato MM/DD/AAAA .

Para cambiar la configuración de idioma en [!DNL Chrome] o cualquier otro navegador, debe modificar la configuración de ese navegador. Los pasos para modificar la configuración de un explorador varían de un explorador a otro. Consulte el [!UICONTROL Ayuda], [!UICONTROL Preferencias]o [!UICONTROL Configuración] para aprender a modificar su configuración.
