# [!DNL Adobe Workfront] para [!DNL iOS]

La aplicación de [!DNL Adobe Workfront] permite acceder al trabajo en cualquier dispositivo [!DNL iOS]. Es posible instalar y usar la aplicación móvil de [!DNL Workfront] en teléfonos y tabletas con iOS 10 o posterior. Hay disponible una aplicación móvil de [!DNL Workfront] dedicada tanto para [!DNL iPhone] como para [!DNL iPad].

Para obtener información sobre cómo iniciar sesión en la aplicación móvil, consulte la sección [Iniciar sesión en la aplicación móvil [!UICONTROL Workfront]](../../../workfront-basics/manage-your-account-and-profile/managing-your-workfront-account/log-in-to-workfront.md#log) en el artículo [Iniciar sesión [!DNL Adobe Workfront]](../../../workfront-basics/manage-your-account-and-profile/managing-your-workfront-account/log-in-to-workfront.md).

## [!UICONTROL Inicio]

El área [!UICONTROL Inicio] le permite trabajar directamente en las tareas, problemas y aprobaciones que se le hayan asignado. En el área [!UICONTROL Inicio] de la aplicación, es posible:

<table style="table-layout:auto"> 
 <col> 
 <col> 
 <tbody> 
  <tr> 
   <td> 
    <ul> 
     <li>Empezar a trabajar en tareas y problemas</li> 
     <li>Marcar tareas y problemas como [!UICONTROL Done]</li> 
     <li>Reasignar tareas y problemas</li> 
     <li>Cambiar [!UICONTROL Commit Date]</li> 
    </ul> </td> 
   <td> 
    <ul> 
     <li>Actualizar estados de tareas y problemas</li> 
     <li>Publicar en el área de Actualizaciones</li> 
     <li>Ver documentos</li> 
     <li>Adjuntar fotos</li> 
     <li> <p>Crear una solicitud o una tarea rápida</p> <p>Nota: Las tareas rápidas creadas en Inicio son tareas personales, a menos que elija asignar un proyecto.</p> </li> 
    </ul> </td> 
  </tr> 
 </tbody> 
</table>

Para obtener más información, consulte los widgets del área de [[!UICONTROL Inicio]](../../../workfront-basics/mobile-apps/using-the-workfront-mobile-app/home-area-widgets-mobile.md) y la sección [[!UICONTROL Mi trabajo] de la aplicación móvil](../../../workfront-basics/mobile-apps/using-the-workfront-mobile-app/my-work-section-mobile.md).

![Área Inicio](assets/mobile-home-area.png)

## [!UICONTROL Tableros]

El área [!UICONTROL Tableros] le permite acceder a sus tableros de [!DNL Workfront].

![Área Tableros](assets/mobile-all-boards-displayed.png)

Para obtener más información, consulte [[!DNL Adobe Workfront] [!UICONTROL Tableros] para dispositivos móviles](/help/quicksilver/workfront-basics/mobile-apps/using-the-workfront-mobile-app/mobile-boards.md).

## [!UICONTROL Proyectos]

El área [!UICONTROL Proyectos] permite interactuar con [!UICONTROL proyectos] de su propiedad y administrarlos, así como con proyectos en los que participe. Desde el área de proyectos de la aplicación, es posible:

<table style="table-layout:auto"> 
 <col> 
 <col> 
 <tbody> 
  <tr> 
   <td> 
    <ul> 
     <li>Actualizar el estado de los proyectos</li> 
     <li>Ver detalles importantes de los proyectos</li> 
     <li>Publicar en el área [!UICONTROL Updates]</li> 
     <li>Convertir tareas, problemas y solicitudes en proyectos</li> 
    </ul> </td> 
   <td> 
    <ul> 
     <li>Interactuar con tareas y problemas de los proyectos</li> 
     <li>Ver documentos</li> 
     <li>Adjuntar fotos</li> 
     <li> <p>Crear tareas rápidas</p> <p>Nota: Las tareas creadas en un proyecto se asignarán automáticamente al proyecto en el que se encuentre. </p> </li> 
    </ul> </td> 
  </tr> 
 </tbody> 
</table>

![Área de proyectos](assets/mobile-projects-area.png)

## Rutas de aprobación

El área [!UICONTROL Aprobaciones] permite ver y aprobar aquellas [!UICONTROL aprobaciones] que se le hayan asignado o delegado. Desde el área de aprobaciones, es posible aprobar:

<table style="table-layout:auto">
 <col>
 <col>
 <tbody>
  <tr>
   <td>
    <ul>
     <li>Trabajo (tareas y problemas)</li>
     <li>Documentos</li>
     <li>Pruebas </li>
    </ul> </td>
   <td>
    <ul>
     <li>Hojas de horas</li>
     <li>Solicitudes de acceso</li>
    </ul> </td>
  </tr>
 </tbody>
</table>

Para obtener más información sobre las aprobaciones, consulte [Aprobaciones en la [!DNL Adobe Workfront] aplicación móvil](../../../workfront-basics/mobile-apps/using-the-workfront-mobile-app/approvals-in-mobile-app.md).

Para obtener información sobre cómo revisar y aprobar pruebas, consulte [Revisar y tomar decisiones sobre pruebas en la [!DNL Adobe Workfront] aplicación móvil](../../../workfront-basics/mobile-apps/using-the-workfront-mobile-app/work-with-proofs-in-mobile-app.md).

## [!UICONTROL Notificaciones]

El área [!UICONTROL Notificaciones] le permite ver y responder notificaciones dentro de la aplicación para su trabajo.

>[!NOTE]
>Los hipervínculos no están disponibles en las notificaciones de actualización, a menos que todo el contenido de una nota sea un hipervínculo sin texto para mostrar.

![Área de notificaciones](assets/mobile-notifications-area.png)

## Hojas de horas

El área [!UICONTROL Plantilla de horas] le permite ver detalles sobre las plantillas de horas. En el área [!UICONTROL Plantillas de horas] de la aplicación, puede:

* Publicar en el área [!UICONTROL Actualizaciones]
* Administrar tipos de hora
* Cambiar el tiempo registrado
* Introducir comentarios en la plantilla de horas
* Cerrar su plantilla de horas

![Área de plantillas de horas](assets/mobile-timesheets-area.png)
