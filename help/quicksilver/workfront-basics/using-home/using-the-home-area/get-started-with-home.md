# Introducción a [!UICONTROL Página principal]

Hay dos maneras de ver su trabajo en la [!UICONTROL Página principal] , ya sea desde el [!UICONTROL Lista de trabajo] o el Calendario.

## Lista de trabajos

La variable [!UICONTROL Lista de trabajo] permite ver todo el trabajo asignado en un solo lugar:

![[!UICONTROL Lista de trabajo] paneles](assets/worklist-and-right-panel-home.png)

A. Utilice el panel izquierdo para filtrar, ordenar y seleccionar el trabajo asignado a usted.

B. Después de seleccionar un elemento de trabajo en el panel izquierdo, utilice el panel derecho para interactuar con campos personalizados, realizar actualizaciones y registrar el tiempo.

Para obtener más información sobre el uso de la variable [!UICONTROL Lista de trabajo en el hogar], consulte los siguientes artículos:

* [Mostrar los elementos en la [!UICONTROL Lista de trabajo] en el [!UICONTROL Página principal] area](../../../workfront-basics/using-home/using-the-home-area/display-items-in-home-work-list.md)
* [Cree elementos de trabajo a partir de [!UICONTROL Página principal] area](../../../workfront-basics/using-home/using-the-home-area/create-work-items-in-home.md)

## Calendario

El calendario le permite visualizar su trabajo mostrando cuándo se deben los elementos de trabajo y permitiendo bloquear el tiempo para completarlos:

![](assets/calendar-home-nwe-350x114.png)

A. Utilice la variable [!UICONTROL Calendario] para crear un mapa visual del trabajo que debe completar. Simplemente haga clic y arrastre los elementos de trabajo desde el [!UICONTROL Lista de trabajo] vaya al Calendario para bloquear el tiempo de trabajo.

B. Utilice la opción de sincronización del Calendario para sincronizar con su [!DNL Outlook] calendario. Administre reuniones y trabaje todo en un solo lugar.

C. Utilice la variable [!UICONTROL Detalles] para abrir un nuevo panel derecho donde puede ver más información sobre el elemento de trabajo.

D. Utilice la barra correspondiente para realizar un seguimiento de cuándo vence el trabajo asignado a usted.

Para obtener más información sobre cómo utilizar el Calendario de inicio, consulte [Utilice la variable [!UICONTROL Calendario principal] ver](../../../workfront-basics/using-home/using-the-home-area/use-home-calendar-view.md).

## Buscar su trabajo

La variable [!UICONTROL Página principal] area es su tienda de ventanilla única para encontrar tareas, problemas y aprobaciones que se le han asignado. Las opciones de filtro y ordenación integradas en la variable [!UICONTROL Página principal] área trabaje juntos para ayudarle a trabajar de la manera que desee. Puede usar la variable [!UICONTROL Ordenar por] para agrupar los elementos y, a continuación, utilizar el filtro para centrarse en el trabajo que debe completarse.

>[!NOTE]
>
>Las opciones de filtro y ordenación se almacenan en el explorador. Si utiliza el mismo explorador de forma consistente en el mismo equipo (y no borra los datos del sitio), los filtros y la ordenación no cambiarán, pero si cambia de explorador o equipo, los filtros y la ordenación serán diferentes.

### Agrupe elementos de trabajo similares junto con el[!UICONTROL Agrupar por]opción &#39;

La variable [!UICONTROL Ordenar por] permite agrupar como elementos en la lista de trabajo. Puede ordenar por:

* [!UICONTROL Finalización planificada]
* [!UICONTROL Inicio planificado]
* [!UICONTROL Fecha de confirmación]
* [!UICONTROL Proyecto]
* [!UICONTROL Mi prioridad]

Para obtener más información sobre cómo utilizar las agrupaciones, consulte [Mostrar los elementos en la [!UICONTROL Lista de trabajo] en el [!UICONTROL Página principal] area](../../../workfront-basics/using-home/using-the-home-area/display-items-in-home-work-list.md).

![](assets/group-by-drop-down-expanded-in-home-with-planned-start-date-nwe-350x273.png)

### Reducir el enfoque con filtros

La variable [!UICONTROL Lista de trabajo] El filtro permite restringir el enfoque a elementos de trabajo específicos.

A continuación se muestran algunos ejemplos de cómo se pueden filtrar elementos que se ven en la página principal:

* Filtrar por tipo de elemento y estado.

   Por ejemplo, si desea mostrar todas las tareas, puede seleccionar la opción [!UICONTROL Tareas] filtro. Si desea ser más específico y mostrar solo las tareas listas para empezar a trabajar, seleccione la opción [!UICONTROL Listo para empezar] filtros en el [!UICONTROL Tareas] filtro.

* Filtrar solo por tipo de elemento.

   Por ejemplo, puede seleccionar [!UICONTROL Problemas] para ver todos los problemas en todos los estados ([!UICONTROL Trabajar con] o [!UICONTROL Solicitado]), o [!UICONTROL Aprobaciones] para ver todos los elementos de trabajo, solicitudes de acceso, parte de horas, documento y aprobaciones de prueba.

* Filtrar solo por estado.

   Por ejemplo, puede seleccionar el [!UICONTROL Completado] para mostrar las tareas y los problemas que se han completado. Esto incluye tareas personales, pero no aprobaciones.

Para obtener más información sobre cómo usar filtros, consulte [Mostrar los elementos en la [!UICONTROL Lista de trabajo] en el [!UICONTROL Página principal] area](../../../workfront-basics/using-home/using-the-home-area/display-items-in-home-work-list.md).

![](assets/displaying-work-items-filters-nwe-350x401.png)

### Acceso al trabajo asignado a su equipo

La variable [!UICONTROL Página principal] tiene una agrupación permanente dedicada a las solicitudes de equipo con un vínculo directo a la página de solicitud del equipo. Utilice esta agrupación para ver y acceder a las solicitudes de cualquiera de los equipos en los que se encuentra.

>[!NOTE]
>
>La variable [!UICONTROL Filtro] y [!UICONTROL Ordenar por] las opciones no afectan a la variable [!UICONTROL Solicitudes del equipo] agrupación. Esta agrupación es visible siempre y cuando tenga trabajo asignado a su equipo.

Para obtener más información sobre el acceso a las solicitudes del equipo, consulte el artículo [[!UICONTROL Administrar] solicitudes de trabajo y equipo en la [!UICONTROL Página principal] area](../../../workfront-basics/using-home/using-the-home-area/manage-work-and-team-requests-home.md).

![](assets/team-requests-expanded-home-group-by-drop-down-nwe-350x314.png)

### Supervisar el trabajo enviado

Realice un seguimiento del trabajo enviado para su aprobación directamente desde [!UICONTROL Lista de trabajo]. Puede recordar al aprobador el trabajo que necesita aprobación. También puede recordar la aprobación si es necesario.

>[!NOTE]
>
>La variable [!UICONTROL Filtro] y [!UICONTROL Ordenar por] las opciones no afectan a la variable [!UICONTROL Aprobaciones enviadas] agrupación. Esta agrupación está visible siempre y cuando tenga trabajo pendiente de aprobación.



![](assets/approvals-expanded-home-group-by-drop-down-nwe-350x415.png)

## Priorizar lo que es importante para usted

La variable [!UICONTROL Lista de trabajo] le permite mostrar y priorizar el trabajo que es importante para usted con la variable [!UICONTROL Mi orden de prioridad por] . Puede agregar hasta 20 elementos a su [!UICONTROL Mi prioridad] lista. Utilice la variable [!UICONTROL Más] para añadir elementos de trabajo a la lista de prioridades.

>[!NOTE]
>
>Cualquier elemento de trabajo agregado a [!UICONTROL Mi prioridad] solo son visibles para usted y nadie puede priorizar el trabajo por usted.

Para obtener más información sobre cómo usar [!UICONTROL Mi prioridad], consulte [Priorizar el trabajo en la variable [!UICONTROL Página principal] area](../../../workfront-basics/using-home/using-the-home-area/prioritize-work-in-home.md).

![](assets/prioritizing-in-home-group-by-drop-down-nwe-350x412.png)

## Hágalo

Utilice el panel derecho en [!UICONTROL Página principal] para actualizar y completar su trabajo.

### Trabaje en un espacio personalizado para usted

Su [!DNL Adobe Workfront] el administrador puede agregar hasta 16 campos personalizados para facilitar la búsqueda y actualización de la información que necesita. La mayoría de los campos son editables, por lo que no tiene que navegar hasta el elemento de trabajo.

Para actualizar cualquiera de los campos de esta área, simplemente haga clic en el campo .

>[!NOTE]
>
>No puede actualizar el [!UICONTROL Fecha de confirmación] en esta área.

![](assets/16-fields-home-350x137.png)

### Actualizar a los colaboradores con información importante

Comuníquese rápida y eficazmente con los compañeros de trabajo de la [!UICONTROL Página principal] . Haces preguntas a los compañeros de trabajo, les notificas de actualizaciones recientes o respondes a comentarios directos. Además, puede actualizar otros sobre el progreso que está realizando:

* Actualización del [!UICONTROL Fecha de confirmación]

   <!--
  <note type="note">
  This is the only place you can update the Commit Date in Home.
  <br>
  </note>
  -->

* Informar sobre cómo sientes que las cosas van
* Actualización de la barra de porcentaje completado

Para obtener más información sobre la comunicación en la variable [!UICONTROL Página principal] área, consulte [Actualizar trabajo](../../../workfront-basics/updating-work-items-and-viewing-updates/update-work.md).

![](assets/updates-home-350x198.png)

### Registra tu tiempo

Haga un seguimiento de las horas que ha trabajado en elementos de trabajo individuales. Puede elegir un tipo de hora general, como [!UICONTROL Hora de la tarea], o bien puede elegir un tipo de hora específico del proyecto configurado por su [!DNL Workfront] administrador.

Para obtener más información sobre cómo iniciar sesión, consulte [!UICONTROL Página principal] , consulte la sección [Página principal](../../../timesheets/create-and-manage-timesheets/log-time.md#home) en el artículo [Tiempo de registro](../../../timesheets/create-and-manage-timesheets/log-time.md).

![](assets/log-time-home-350x181.png)
