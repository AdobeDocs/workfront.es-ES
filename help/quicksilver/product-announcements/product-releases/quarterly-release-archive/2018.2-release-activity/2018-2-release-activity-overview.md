### Finaliza la compatibilidad con TLS {#tls-support-is-ending}

Para proporcionar una seguridad óptima, Workfront requiere que todas las integraciones y el tráfico web que utilizan TLS 1.0 o versiones anteriores se actualicen para utilizar TLS 1.1 o versiones posteriores.

Para obtener más información, consulte [Se requiere TLS 1.2 en Adobe Workfront](../../../../product-announcements/announcements/announcement-archive/tls-1-disabled.md).

### Eliminación de aplicación de Flash {#flash-application-removal}

Todas las funcionalidades que dependen de la tecnología de Flash se eliminarán de Workfront con la versión 2018.3 y se sustituirán por una nueva solución.

Esta funcionalidad incluye:

* El gráfico Gantt anterior se reemplazará con el nuevo

  Para obtener más información, vea [Ver información en el diagrama de Gantt](../../../../manage-work/gantt-chart/use-the-gantt-chart/view-info-in-gantt.md).

* Portfolio Optimizer se reemplazará con el nuevo Portfolio Optimizer

  Para obtener más información, consulte [Descripción general de Portfolio Optimizer](../../../../manage-work/portfolios/portfolio-optimizer/portfolio-optimizer-overview.md).

* El Planificador de capacidades será reemplazado con el nuevo Planificador de recursos

  Para obtener más información, consulte [Planificación de recursos en Adobe Workfront](../../../../resource-mgmt/resource-planning/resource-planning-overview.md).

* El Administrador de presupuesto será reemplazado con el nuevo Planificador de recursos

  Para obtener más información, consulte [Planificación de recursos en Adobe Workfront](../../../../resource-mgmt/resource-planning/resource-planning-overview.md) [.](https://support.workfront.com/hc/en-us/sections/115001049327)

* La cuadrícula de recursos se reemplazará con el nuevo Planificador de recursos

  Para obtener más información, consulte [ [Planificación de recursos en Adobe Workfront](../../../../resource-mgmt/resource-planning/resource-planning-overview.md).](https://support.workfront.com/hc/en-us/sections/115001049327)

* Las estimaciones de recursos del caso comercial se reemplazarán con la nueva sección de presupuesto de recursos del caso comercial en el Planificador de recursos

* El Generador de equipos heredado se reemplazará con la nueva área Programación de recursos

  Para obtener más información, consulte Introducción a la programación de recursos.

### Finaliza la compatibilidad con complementos de Microsoft Outlook {#microsoft-outlook-add-in-support-is-ending}

Workfront dejará de ser compatible con la versión de escritorio del complemento después de la versión de producto 18.2 que se lanzará en julio.

Ahora es el momento de cambiar a Workfront para Outlook y convertirse en un centro de productividad. En lugar de proporcionar dos complementos distintos para Outlook, ahora proporcionamos un complemento universal que se puede utilizar en cualquier entorno: en un equipo, Mac, escritorio, Web e incluso en la aplicación móvil de Outlook.

Al instalar el nuevo Workfront para Outlook en cualquiera de sus entornos, su equipo puede hacer más, mantenerse organizado y priorizar su trabajo en tiempo real.

Utilice los siguientes recursos para obtener más información sobre Workfront para Outlook y empezar:

* [Sitio web y vídeo](https://www.workfront.com/integrations/outlook) 
* [Uso de Workfront con Outlook](../../../../workfront-integrations-and-apps/using-workfront-with-outlook/workfront-for-outlook.md) (artículo de ayuda)

### Eliminación de versiones de API antiguas {#removing-old-api-versions}

Como parte de nuestros esfuerzos continuos por mejorar nuestra API, estamos eliminando las versiones de API antiguas que ya no son compatibles con las versiones más recientes de Workfront.

Consulte la [programación de versiones y soporte de API](../../../../wf-api/api/api-version-support-schedule.md) para obtener más información.
