### Finaliza la compatibilidad con TLS {#tls-support-is-ending}

Para proporcionar una seguridad óptima, Workfront requiere que todas las integraciones y tráfico web que utilicen TLS 1.0 o una versión anterior se actualicen para utilizar TLS 1.1 o posterior.

Para obtener más información, consulte [Se requiere TLS 1.2 en Adobe Workfront](../../../../product-announcements/announcements/announcement-archive/tls-1-disabled.md).

### Eliminación de la aplicación de Flash {#flash-application-removal}

Todas las funciones que dependen de la tecnología de Flash se eliminarán de Workfront con la versión 2018.3 y se sustituirán por una nueva solución.

Esta funcionalidad incluye:

* El diagrama de Gantt heredado se reemplazará por el nuevo diagrama de Gantt

   Para obtener más información, consulte [Ver información en el diagrama de Gantt](../../../../manage-work/gantt-chart/use-the-gantt-chart/view-info-in-gantt.md).

* El optimizador de Portfolio se reemplazará con el nuevo optimizador de Portfolio

   Para obtener más información, consulte [Información general sobre Portfolio Optimizer](../../../../manage-work/portfolios/portfolio-optimizer/portfolio-optimizer-overview.md).

* El planificador de capacidad se reemplazará con el nuevo planificador de recursos

   Para obtener más información, consulte [Planificación de recursos en Adobe Workfront](../../../../resource-mgmt/resource-planning/resource-planning-overview.md).

* El Administrador de presupuesto se reemplazará con el nuevo Planificador de recursos

   Para obtener más información, consulte [Planificación de recursos en Adobe Workfront](../../../../resource-mgmt/resource-planning/resource-planning-overview.md) [.](https://support.workfront.com/hc/en-us/sections/115001049327)

* La cuadrícula de recursos se reemplazará con el nuevo planificador de recursos

   Para obtener más información, consulte [ [Planificación de recursos en Adobe Workfront](../../../../resource-mgmt/resource-planning/resource-planning-overview.md).](https://support.workfront.com/hc/en-us/sections/115001049327)

* Las estimaciones de recursos del caso empresarial se sustituirán por la nueva sección de presupuestación de recursos del caso empresarial del planificador de recursos

   <!--
  <p data-mc-conditions="QuicksilverOrClassic.Draft mode">For more information, see <a href="../../../../resource-mgmt/resource-mgmt-overview/migrate-resource-estimates-to-budgeting.md" class="MCXref xref" xrefformat="{para}">Migrate from Legacy Resource Estimates to Resource Budgeting </a>.</p>
  -->

* El Generador de equipos heredado se reemplazará por el nuevo área de programación de recursos

   Para obtener más información, consulte &quot;Introducción a la programación de recursos&quot;.

   >[!NOTE]
   Las herramientas de programación de recursos se han desaprobado y eliminado de Workfront con la versión 23.1. Para obtener información sobre la programación de recursos mediante el equilibrador de carga de trabajo, consulte [Información general del equilibrador de carga de trabajo](../../../../resource-mgmt/workload-balancer/overview-workload-balancer.md).

* El visor de pruebas heredado se reemplazará con el nuevo visor de pruebas web y visor de pruebas de escritorio.

   En la versión 2018.2, el visor de pruebas heredadas ya no es el visor predeterminado.

   Para obtener más información, consulte los siguientes recursos:

   * [Revisión de pruebas en el visor de pruebas web](https://support.workfront.com/hc/en-us/sections/115000275214)
   * [Revisión de pruebas en el visualizador de pruebas de escritorio](https://support.workfront.com/hc/en-us/sections/360000686434)

### Finaliza la compatibilidad con el complemento de Microsoft Outlook {#microsoft-outlook-add-in-support-is-ending}

Workfront dejará de ser compatible con la versión de escritorio del complemento después de la versión de producto 18.2 publicada en julio.

¡Ahora es el momento de cambiar a Workfront para Outlook y convertirse en un motor de productividad! En lugar de proporcionar dos complementos distintos para Outlook, ahora se proporciona un complemento universal que se puede utilizar en cualquier entorno: en un equipo, Mac, escritorio, web e incluso en la aplicación móvil de Outlook.

Al instalar el nuevo Workfront para Outlook en cualquiera de sus entornos, su equipo puede hacer más, mantenerse organizado y priorizar su trabajo en tiempo real.

Utilice los siguientes recursos para obtener más información sobre Workfront para Outlook y comenzar:

* [Sitio web y vídeo](https://www.workfront.com/integrations/outlook) 
* [Uso de Workfront con Outlook](../../../../workfront-integrations-and-apps/using-workfront-with-outlook/workfront-for-outlook.md) (artículo de ayuda)

### Eliminación de versiones de API antiguas {#removing-old-api-versions}

Como parte de nuestros esfuerzos continuos para mejorar nuestra API, eliminamos las versiones antiguas de API que ya no son compatibles con las versiones más recientes de Workfront.

Consulte la [Programación de versiones y asistencia de API](../../../../wf-api/api/api-version-support-schedule.md) para obtener más información.
