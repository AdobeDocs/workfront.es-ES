## Mejoras para todos los usuarios

Algunas funciones se lanzaron antes de la versión 2019.4. Para ver qué se publicó antes de la versión 2019.4, consulte [Otras mejoras de 2019.4](../../../../product-announcements/product-releases/quarterly-release-archive/2019.4-release-activity/2019.4-other-enhancements.md).

La versión 2019.4 contiene las siguientes mejoras:

* [Mejoras en la lista](#list-enhancements)
* [Mejoras en dispositivos móviles](#mobile-enhancements)
* [Mejoras en la programación de recursos](#resource-scheduling-enhancements)
* [Otras mejoras](#other-enhancements)

### Mejoras en la lista {#list-enhancements}

>[!IMPORTANT]
>

Estas mejoras se eliminaron del entorno Vista previa el 8 de noviembre de 2019 y no se incluirán en la versión 2019.4 para producción. Estas mejoras se volverán a añadir al entorno de Vista previa después de la versión 2019.4 a Producción y estarán disponibles para Producción en una versión futura.

A continuación se muestra la cronología de eliminación para estas opciones de SSO:

* Asistencia limitada: De agosto de 2018 a enero de 2019

   Se solucionarán los principales problemas presentados durante este tiempo; no se solucionarán los problemas de baja gravedad.

* Obsolescencia: De enero de 2019 a noviembre de 2019 (con la versión de 2019.4)

   No compatible.

* Ya no está disponible en Workfront para nuevas configuraciones: Noviembre de 2019
* Migración necesaria para clientes existentes: principios de 2020

### La compatibilidad con TLS 1.0 y 1.1 finaliza {#tls-1-0-and-1-1-support-is-ending}

Para proporcionar una seguridad óptima, Workfront requiere que todas las integraciones y tráfico web que utilicen TLS 1.1 o una versión anterior se actualicen para utilizar TLS 1.2. En el entorno de vista previa, TLS 1.0 ya está deshabilitado.
