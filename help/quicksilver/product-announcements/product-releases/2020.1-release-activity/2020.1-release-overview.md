A continuación se muestra la cronología de eliminación para estas opciones de SSO:

* Asistencia limitada: De agosto de 2018 a enero de 2019

   Se solucionarán los principales problemas presentados durante este tiempo; no se solucionarán los problemas de baja gravedad.

* Obsolescencia: De enero de 2019 a noviembre de 2019 (con la versión de 2019.4)

   No compatible.

* Ya no está disponible en Workfront para nuevas configuraciones: Noviembre de 2019
* Migración necesaria para clientes existentes: principios de 2020

### La compatibilidad con TLS 1.0 y 1.1 finaliza {#tls-1-0-and-1-1-support-is-ending}

Para proporcionar una seguridad óptima, Workfront requiere que todas las integraciones y tráfico web que utilicen TLS 1.1 o una versión anterior se actualicen para utilizar TLS 1.2. En el entorno de vista previa, TLS 1.0 ya está deshabilitado.

Para obtener más información, consulte [Se requiere TLS 1.2 en Adobe Workfront](../../../product-announcements/announcements/announcement-archive/tls-1-disabled.md).

### Eliminación de la aplicación de Flash {#flash-application-removal}

Todas las funciones que dependen de la tecnología de Flash se han eliminado de Workfront desde la versión 2018.3 y se han sustituido por una nueva solución.

Para obtener información detallada sobre las herramientas de Flash heredadas y sus reemplazos, consulte [Sustitución de herramientas basadas en Flash en Adobe Workfront](../../../product-announcements/announcements/announcement-archive/replace-flash-tools.md).
