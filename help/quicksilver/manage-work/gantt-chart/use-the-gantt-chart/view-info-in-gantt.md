## Busque el  [!UICONTROL Gráfico Gantt]

Puede localizar tanto el gráfico Gantt de lista de tareas como la lista de proyectos [!UICONTROL Gráfico Gantt] desde varias áreas dentro de Workfront. Para obtener más información, consulte [Introducción a la [!UICONTROL Gráfico Gantt]](../../../manage-work/gantt-chart/use-the-gantt-chart/get-started-with-gantt.md).

## Ver tareas en [!UICONTROL Ruta crítica]

En la lista de proyectos [!UICONTROL Gráfico Gantt], tareas que no están en la [!UICONTROL Ruta crítica] mostrar como líneas horizontales de color azul claro. Tareas que se encuentran en la [!UICONTROL Ruta crítica] de un proyecto se muestran como líneas horizontales rojas.

Para obtener más información sobre las tareas de [!UICONTROL Ruta crítica], consulte [Información general del proyecto [!UICONTROL Ruta crítica]](../../../manage-work/tasks/manage-tasks/critical-path.md).

## Ver información de tareas en la lista de proyectos [!UICONTROL Gráfico Gantt]

Puede ver la información de tareas de un proyecto directamente desde la Lista de proyectos. Las tareas se enumeran con el nombre de cada proyecto.

>[!NOTE]
>
>No puede editar tareas desde la Lista de proyectos [!UICONTROL Gráfico Gantt].

Puede ver la información de tareas de un proyecto directamente desde una lista de proyectos en las áreas siguientes:

* En el [!UICONTROL Proyectos] área
* Dentro de un Portfolio
* Dentro de un programa

Para ver las tareas de un proyecto a partir de una lista de proyectos:

1. Vaya a una de las áreas mencionadas anteriormente.

   Por ejemplo, desde el [!UICONTROL Menú principal], haga clic en **[!UICONTROL Proyectos]**.

   Se muestra una lista de proyectos.

1. Haga clic en **[!UICONTROL Gráfico Gantt]** icono ![](assets/gantt-icon-nwe.png) en la esquina superior derecha de la pantalla.

   <!--
   <p data-mc-conditions="QuicksilverOrClassic.Draft mode">(NOTE: images conditioned for classic and nwe) <br></p>
   -->

1. Haga clic en **[!UICONTROL Mostrar lista de tareas]** icono.

1. En la lista de proyectos de la izquierda, haga clic en la flecha desplegable situada junto al nombre del proyecto para mostrar cualquier tarea debajo de ese proyecto.\
   Muestra información de la tarea en  [!UICONTROL Gráfico Gantt].\
   ![Show_task_list_enabled_project_expand.png](assets/show-task-list-enabled-project-expanded-350x78.png)

1. (Opcional) Haga clic en **[!UICONTROL Imprimir]** en la esquina superior derecha para exportar el [!UICONTROL Gráfico Gantt].

   >[!NOTE]
   >
   >La lista de proyectos [!UICONTROL Gráfico Gantt] solo exporta proyectos. No se incluye la información de la tarea.

## Cambie el período de tiempo para el que se muestra información en la [!UICONTROL Gráfico Gantt]

Puede ajustar el periodo de tiempo que se muestra en la [!UICONTROL Gráfico Gantt] para mostrar información en un nivel granular, o puede navegar rápidamente a una vista de día, semana, mes, trimestre o año:

* [Cambiar el período de tiempo en un nivel granular](#change-the-time-period-on-a-granular-level)
* [Ver información por día, semana, mes, trimestre o año](#view-information-by-day-week-month-quarter-or-year)

### Cambiar el período de tiempo en un nivel granular {#change-the-time-period-on-a-granular-level}

1. Pase el ratón sobre la cronología de  [!UICONTROL Gráfico Gantt]A continuación, arrastre el indicador de zoom de izquierda a derecha para expandir o contraer la cronología.\
   ![zoom_tool_in_gantt.png](assets/zoom-tool-in-gantt-350x180.png)

### Ver información por día, semana, mes, trimestre o año {#view-information-by-day-week-month-quarter-or-year}

1. En el  [!UICONTROL Gráfico Gantt], haga clic en el menú desplegable lapso de tiempo.

   ![](assets/timeline-options.png)

1. Seleccione un lapso de tiempo entre las siguientes opciones disponibles:

   * **[!UICONTROL Ajustar todo]**: Esta opción muestra la línea de tiempo de todo el proyecto.
   * **[!UICONTROL Todos los proyectos]**: esta opción solo está disponible en el gráfico Gantt de lista de proyectos.
   * **[!UICONTROL Año]**
   * **[!UICONTROL Trimestre]**
   * **[!UICONTROL Mes]**
   * **[!UICONTROL Semana]**
   * **[!UICONTROL Día]**

1. (Opcional) Seleccione un lapso de tiempo más granular, como [!UICONTROL Semana] o [!UICONTROL Día]y haga clic y arrastre la barra de desplazamiento horizontal en la parte inferior de la  [!UICONTROL Gráfico Gantt] para desplazarse de izquierda a derecha en la línea de tiempo del proyecto.\
   Una instantánea de la cronología de [!UICONTROL Gantt] se muestra para mostrar todo el proyecto.

   >[!TIP]
   >
   >La instantánea de la cronología solo se muestra después de hacer clic en la barra de desplazamiento horizontal.

   ![stretchy_gantt_minimap_with_outline__1_.png](assets/stretchy-gantt-minimap-with-outline--1--350x140.png)

1. (Opcional) Haga clic en cualquier lugar dentro de la instantánea de la línea de tiempo para desplazarse a un punto específico de la vida del proyecto.\
   O\
   Arrastre los controles del visor de instantáneas para seleccionar un intervalo de línea de tiempo específico y mostrarlo en la pantalla principal [!UICONTROL Gantt].

## Uso de filtros, vistas y agrupaciones

El [!UICONTROL Gráfico Gantt] es una representación visual de la información que se muestra actualmente en la lista de tareas. Puede aplicar filtros, vistas y agrupaciones a los objetos enumerados en ambos [!UICONTROL Gráfico Gantt]s.

>[!CAUTION]
>
>No puede aplicar filtros, vistas y agrupaciones al seleccionar [!UICONTROL Manual] guardar [!UICONTROL Planificación de calendario] para guardar los cambios en la lista de tareas. Para obtener información sobre cómo guardar cambios en tareas de una lista, consulte [Edición de tareas en una lista](../../../manage-work/tasks/manage-tasks/edit-tasks-in-a-list.md).

Los filtros y agrupaciones que aplique a la lista se reflejan tanto en la Lista de proyectos como en la Lista de tareas  [!UICONTROL Gráfico Gantt]Las y también se incluyen cuando se exportan los gráficos Gantt:

* Filtros\
   Puede aplicar un Filtro a la lista para controlar la información que se muestra en la [!UICONTROL Gráfico Gantt].\
   Para obtener información sobre la aplicación de un filtro, consulte  [Resumen de filtros](../../../reports-and-dashboards/reports/reporting-elements/filters-overview.md).

* Agrupaciones\
   Las agrupaciones que aplique a la lista se reflejan en la variable [!UICONTROL Gráfico Gantt].\
   Para obtener información sobre la aplicación de una agrupación, consulte  [Información general sobre agrupaciones en Adobe Workfront](../../../reports-and-dashboards/reports/reporting-elements/groupings-overview.md).

Las vistas no se reflejan en [!UICONTROL Gráfico Gantt]. Sin embargo, al exportar el [!UICONTROL Gráfico Gantt] (como se describe en  [Exporte el [!UICONTROL Gráfico Gantt] al PDF](../../../manage-work/gantt-chart/use-the-gantt-chart/export-gantt-chart-to-pdf.md)), se exporta la lista de tareas además de [!UICONTROL Gráfico Gantt], con la vista actual aplicada a la lista.

## Configurar opciones de visualización

Puede elegir qué tipo de información se muestra en ambos [!UICONTROL Gráficos Gantt]. Para obtener más información, consulte [Configure cómo se muestra la información en la [!UICONTROL Gráfico Gantt]](../../../manage-work/gantt-chart/use-the-gantt-chart/configure-info-on-gantt-chart.md).
