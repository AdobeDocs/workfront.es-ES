# Programación de recursos para proyectos

<!--Audited: 5/2025-->

Programe recursos para un solo proyecto para indicar el trabajo al que esté asignado cada recurso, así como el tiempo que se debería dedicar para completar el trabajo. Puede programar recursos para un proyecto y asignar horas a cada recurso mediante el Distribuidor de cargas de trabajo en el nivel de proyecto.

## Programación de recursos para proyectos

Puede utilizar el Distribuidor de cargas de trabajo para programar recursos para un proyecto.

1. Vaya al proyecto para el que quiera programar recursos.
1. Haga clic en **Distribuidor de cargas de trabajo** en el panel de la izquierda.

   Para obtener información sobre la programación de recursos mediante el Distribuidor de cargas de trabajo, consulte los siguientes artículos:

   * [Información general sobre la asignación de trabajo en el Distribuidor de cargas de trabajo](../../../resource-mgmt/workload-balancer/assign-work-in-workload-balancer.md)
   * [Administrar asignaciones de usuario en el equilibrador de carga de trabajo](../../../resource-mgmt/workload-balancer/manage-user-allocations-workload-balancer.md)
