# Campos de finanzas de proyecto

Realizar un seguimiento de las finanzas de los proyectos es fundamental para administrar los proyectos.

Como administrador de proyectos, puede actualizar manualmente algunos campos financieros. Adobe Workfront calcula automáticamente otros datos financieros.

Puede realizar un seguimiento de las finanzas mediante varios campos que capturan el presupuesto, el coste y los ingresos en Workfront.

Para obtener más información, consulte también los siguientes artículos:

* [Glosario de terminología de Adobe Workfront](../../../workfront-basics/navigate-workfront/workfront-navigation/workfront-terminology-glossary.md)
* [Finanzas del proyecto](../../../manage-work/projects/project-finances/project-finances-overview.md)
* [Administrar información en el área de finanzas del proyecto](../../../manage-work/projects/project-finances/manage-project-finance-area.md)
* [Administrar finanzas de tarea en la sección Detalles de la tarea](../../../manage-work/tasks/manage-tasks/task-finances-in-details.md)

## Cómo calcula Workfront las finanzas

Puede realizar un seguimiento de los campos financieros en el nivel de proyecto y tarea en Workfront. Puede realizar un seguimiento de los campos financieros limitados para detectar problemas.
